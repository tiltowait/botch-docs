**Botch** and **Beat** use a web-based wizard for character creation. No login is necessary; simply enter the command in Discord and click the generated link. Once active, the link is available for **20 minutes**.

Before starting, be sure to [install the bot](/guides/get-started)!

### Steps

<Steps>

1. Start the character creation wizard with `/character wizard`. Select the `era` appropriate for your chronicle.

    <Picture
      src={wizardCommand}
      alt="Use /character wizard command"
      width={imgWidth}
    />

1. Click the generated link.

    <Picture
      src={wizardLink}
      alt="Click the link"
      width={imgWidth}
    />

1. Select character type.

    <Picture
      src={selectCharType}
      alt="Select character type"
      width={imgWidth}
    />

1. Enter details and click submit!

    <Picture
      src={enterDetails}
      alt="Enter character details"
      width={imgWidth}
    />

    <Picture
      src={created}
      alt="Character created"
      width={imgWidth}
    />

</Steps>

### Finished!

You're all done! You can close the browser tab. Proceed to the next session.
