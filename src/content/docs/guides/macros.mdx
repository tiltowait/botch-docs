Macros allow you to perform common [rolls](/guides/rolls) more quickly. For example, if you often roll `Strength + Brawl.Biting` for your Vampire's hunts, you can create a macro called *hunt* that makes that same roll. Combined with [roll comments](/reference/macros#macro-create), you can save a lot of typing!

## Create

<Tabs syncKey="bot">
  <TabItem label="Botch (WoD)">
    <Code code="/macro create name:hunt pool:Strength+Brawl.Biting difficulty:6" mark={macroHighlights} />

    The above command will create a macro called `hunt`, with a pool using `Strength` and `Brawl`, plus a `Biting` specialty, at Difficulty 6.
  <Picture
    src={botchMacroCreate}
    width={imgWidth}
    alt="Botch: /macro create"
    densities={[4]}
  />
  </TabItem>
  <TabItem label="Beat (CofD)">
    <Code code="/macro create name:hunt pool:Strength+Brawl.Biting" mark={macroHighlights} />

    The above command will create a macro called `hunt`, with a pool using `Strength` and `Brawl`, plus a `Biting` specialty.
  <Picture
    src={beatMacroCreate}
    width={imgWidth}
    alt="Beat: /macro create"
    densities={[4]}
  />
  </TabItem>
</Tabs>

<Aside type="tip">
  You can use [Shortcuts](/guides/rolls#shortcuts) in your macro pools. They will be expanded at macro creation, so there's no chance of ambiguity if you later assign some [Custom Traits](/guides/traits#assign).
</Aside>
